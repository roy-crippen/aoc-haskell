# aoc2024

[Advent of Code 2024](https://adventofcode.com/2024/) in Haskell

## run all days

```bash
$ cabal run
```

## repl for a particular day

```bash
# DayXX where XX is the day number
# example for day 1
$ cabal repl
> :l Day01
```

## repl for a particular day using example data

```bash
# DayXX where XX is the day number
# example for day 1
$ cabal repl --ghc-options="-DEXAMPLE"
> :l Day01
```
